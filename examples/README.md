# Examples

This directory contains example scripts that demonstrate how to use the SSH emulator.

## Available Examples

- `paramiko_example.py`: Demonstrates how to connect to the emulator using the Paramiko SSH client.

## Running Examples

1. Start the SSH emulator in a separate terminal:
   ```bash
   python -m emulator
   ```

2. Run the example script in another terminal:
   ```bash
   python examples/paramiko_example.py
   ```
